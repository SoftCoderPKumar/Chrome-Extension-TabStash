## TabStash - Chrome Extension

TabStash is a simple and lightweight Chrome extension that helps you save and manage your important tabs and URLs in one place. It works like a smart bookmarking tool so you can access your links later without losing them.

## 🚀 Features

Save the current active tab instantly

Save all open tabs with one click

Add any URL manually to the list

Open a single saved URL in a new tab

Open all saved URLs in individual tabs

Delete links one by one

Clear the entire saved list

Clean and easy-to-use interface

## 📌 Use Cases

Save research links for later

Store important work tabs

Manage study or project resources

Avoid losing important tabs when closing the browser

## 🛠️ Installation

### From Source (Developer Mode)

Download or clone this repository

Open Chrome and go to: `chrome://extensions/`

Enable **Developer Mode** (top right)

Click **Load Unpacked**

Select the project folder

TabStash will be added to your browser

## 📖 How to Use

### Save Current Tab

Click on the "Save Current Tab" button to store the active tab in the list.

### Save All Tabs

Click on "Save All Tabs" to store all open tabs at once.

### Add URL Manually

Paste any URL into the input field and click "Add" to save it.

### Open Saved Links

Click on a link to open it in a new tab

Use "Open All" to open all saved links in separate tabs

### Delete Links

Remove a single link using the delete option

Use "Delete All" to clear the entire list

## 💾 Data Storage

TabStash uses Chrome local storage to securely save your URLs, so your data remains available even after restarting the browser.

## 🔒 Permissions Used

`tabs` – To access and save open tabs

`storage` – To store saved URLs locally
